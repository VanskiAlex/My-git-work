1. Какой тип хранения данных использует GIT. В чем его преимущества?

GIT для хранения данных использует репозитории, которые хранятся на всех клиентских компьютерах, что защищает от потерь данных и позволяет полноценно управлять версиями проекта оффлайн.

2. Какой командой нужно воспользоватся чтобы узнать свой user.neme и user.email в git bash?

git config user.name
git config user.email

3. Какой последовательностью команд нужно воспользоватся чтобы создать локальный репозиторий и добавить в него файлы?

git init
git add .

4. Какая последовательность команда добавит новый файл index коммита?

git add .index.txt
git commit -m "Первый коммит"
git push

5. Какая команда загрузит новый репозиторий c github.com? что предварительно нужно сделать на сервере github.com.

Создать новый репозиторий на сервере через WEB интерфейс.
git clone склонировать созданный репозиторий на компьютер


6. Как создать новый репозиторий на github.com из локальной консоли git bash?

 git init
 git add 
 git commit -m 'комментарий'
 git push
